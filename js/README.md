# Personal Birthday Website

A modern, minimal, and interactive birthday website for celebrating on December 31st.

## Features

1. **Landing Section**
   - Birthday announcement with large date display
   - Interactive countdown timer to the birthday

2. **Anonymous Birthday Messages**
   - Form for visitors to leave birthday wishes
   - Messages stored locally in browser
   - Display of recent messages

3. **Interactive Quiz**
   - 7 questions about the birthday person
   - Score tracking and results display
   - Fun way for visitors to test their knowledge

4. **Year in Review**
   - Timeline of highlights from the past year
   - Clean, visually appealing layout

5. **Responsive Design**
   - Works on mobile, tablet, and desktop
   - Accessible navigation

## Setup Instructions

1. **Clone or download** the project files
2. **Open `index.html`** in your web browser
3. **Customize** the content in each file:
   - Update quiz questions in `js/quiz.js`
   - Modify year-in-review content in `index.html`
   - Adjust colors in `css/style.css` if desired

## Customization Guide

### Changing Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary: #FF6B6B;     /* Main color */
    --secondary: #4ECDC4;   /* Secondary color */
    --accent: #FFD166;      /* Accent color */
}
